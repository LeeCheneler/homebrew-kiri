# homebrew-kiri

Homebrew tap for [kiri](https://github.com/LeeCheneler/kiri) — a local-first, git-based workflow orchestrator for personal automation.

## Install

```sh
brew install LeeCheneler/kiri/kiri
```

This auto-taps `LeeCheneler/homebrew-kiri` on first install. To upgrade later:

```sh
brew upgrade kiri
```

## Updates

The formula in this repo is auto-bumped on every published release of kiri by the [release workflow](https://github.com/LeeCheneler/kiri/blob/main/.github/workflows/release.yml). Manual edits get overwritten on the next release — open issues against [LeeCheneler/kiri](https://github.com/LeeCheneler/kiri) instead.
